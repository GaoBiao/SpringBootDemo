# SpringBootDemo
基于SpringBoot的webDemo<br>
IDE:MyEclipse 2017<br>
JAVA环境:jdk8<br>
项目管理工具:maven
技术关键词：Spring Boot,Spring MVC,Spring Data,Spring Security,Hibernate,EhCache,MySql

子模块运行前，WebDemoDomain模块需先执行mvn install命令
